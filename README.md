## Getting Started

### Prerequisites
- Python 3.11+
- Git

### Installation
1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
